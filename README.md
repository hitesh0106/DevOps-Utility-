# DevOps Utility Hub

![DevOps Utility Hub](https://img.shields.io/badge/Status-Active-success) ![License](https://img.shields.io/badge/License-MIT-blue) ![100% Client-Side](https://img.shields.io/badge/Type-100%25%20Client--Side-orange)

**DevOps Utility Hub** is a comprehensive, open-source collection of 12+ essential tools tailored for DevOps engineers, sysadmins, and developers. Built with pure HTML, CSS, and JavaScript, everything runs completely in your browser—meaning **zero backend, 100% privacy, and instant results**.

## 🚀 Available Tools

Here is the complete list of tools currently available in the DevOps Utility Hub:

1. **YAML Validator:** Validate & format YAML with detailed error reporting
2. **JSON Validator:** Validate, format & minify JSON with ease
3. **YAML ↔ JSON Converter:** Convert between YAML and JSON formats instantly
4. **Dockerfile Generator:** Generate production-ready Dockerfiles
5. **Compose Generator:** Create Docker Compose files with ease
6. **Base64 Encoder:** Encode & decode Base64 for K8s secrets
7. **Password Generator:** Generate secure passwords & secrets
8. **.gitignore Generator:** Create .gitignore for any tech stack
9. **Nginx Config Generator:** Generate Nginx configurations quickly
10. **Subnet Calculator:** Calculate subnets, IPs, and CIDR ranges
11. **Cron Expression Builder:** Build & understand cron expressions
12. **JWT Decoder:** Decode and inspect JWT tokens
13. **Regex Tester:** Test regular expressions in real-time

## 🛠️ How to Run Locally

Since this toolkit operates entirely client-side, running it locally is incredibly fast and simple.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/hitesh0106/DevOps-Utility-.git
   ```

2. **Open directly:**
   Navigate into the project folder and simply open `index.html` in your favorite web browser.
